# Markdown
Cria template de documentação de tabelas em markdown.


## **Dependencies**

```
1. import snakemd
2. import re
3. import unidecode
4. import pandas
```
## Repository template

```
C:
└───markd_table
    ├───inputs
    │   └───tables
    └───outputs
```

## **How it works?**

1-Insert or create your tables files in `./input/tables`.
2-Run the main file in your prompt command.
3-Check your new markdown file.
